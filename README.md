## 🌍 Why I Build What I Build

African organisations face the same threat actors as the rest of the world — ransomware, business email compromise, insider threats — but with a fraction of the security budget and almost no local talent pipeline. I build practical, open-source-first tooling because expensive vendor stacks aren't the answer for this market. Data sovereignty matters too: local LLM deployment means sensitive security data never leaves the organisation.

ARX AMANI exists to bridge that gap for Nigerian SMEs. The PhD research I'm pursuing exists to advance it further.
